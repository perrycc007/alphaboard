⭐ 7.1 第一部分：Stage Theory（最重要的基礎）
⭐ 技術分析不是「買低賣高」
⭐ 技術分析是「跟趨勢賺錢、在反轉時離場」

🔵 Stage 1 — 底部築基期（Sideways）

無量

無趨勢

📌 適用對象：

🟢 Stage 2 — 上升趨勢（強勢期）
👉 所有最強的股票、ETF、行業都在這段時間爆發
📌 特徵：
股價站上 50MA、150MA、200MA


MA 全部向上


量能上升


領導股爆發


行業輪動強


Breakout 成功率最高

📌 心法：

✔ 一齊掂到 key level，再有同樣反應

⭐ 7.2.2 Group Setup：Long / Short 兩種「板塊同形」
Short（板塊轉弱常見形態）：
• 低 volume rally 上去撞 20EMA / 50SMA / 200SMA 阻力
• Double top
• Benchmark 心理位（$50 / $100 / $300 / $1000）附近轉弱
Long（板塊轉強常見形態）：
• 低 volume 拉回撞 20EMA / 50SMA / 200SMA 支持
• Undercut & Rally：micro（3 個月內）或 macro（幾個月到 1–2 年）前低跌穿後即刻收回
實戰應用：如果你見到同一個 group（例如 semiconductor）有 3 隻以上股票準備撞 resistance，咁你可以 ready short 呢個 group（反之亦然）。
Form 咗 bias 之後，就等佢 hit key level，再入場：
• 流動性細（avg volume < 10 萬）：可以直接 entry + set 1–3% stop
• 流動性大（avg volume ≥ 100 萬）：建議用 620 chart 做精準入場

⭐ 7.3.1 Cheat / Low Cheat / Cheat Area（點解唔一定要 Relative Volume）
解釋點解普通 breakout at new high（破新高）通常要 rel vol 去 prove 大戶入貨：
因為好多人淨係識 trade「破新高」，而好多 algo program 好鍾意喺嗰個位做假突破洗走散戶。冇 volume 支持嘅 breakout，成功率會更低。
但 cheat area / low cheat 係另一回事：因為識 trade 呢兩種 entry 嘅人比較少，市場上冇咁大誘因特登寫 program 去洗走一班本身數量唔多、又有技術嘅人，所以相對上唔需要把 relative volume 當成必須條件（有就更好）。

⭐ 7.3 第三部分：VCP（Volatility Contraction Pattern）變異收窄
這是 Mark Minervini 的神技。
 你一定要教。

🔵 什麼是 VCP？
⭐「上升時的波幅逐漸縮小 → 供應減少 → 隨時爆升」
像：
(波幅大) ↓
 (波幅中) ↓
 (波幅小) ↓
 (準備突破) → 🚀
這代表：
✔ 賣家愈來愈少
 ✔ 買家愈來愈多
 ✔ 一突破就會爆上去

🔶 VCP 的 4 大特徵
趨勢向上


3–5 次波幅收窄


成交量隨波幅縮小


最後一次回調最弱



🟢 為什麼 VCP 這麼強？
因為：
⭐「賣家已經全部賣清」
⭐「突破時只剩買家」
⭐ 7.7.1 620 Chart（Key Level 附近的精準入場法）
用途：當你已經有 bias（long/short）+ key level，620 chart 係用嚟「揾 timing」入場。
設定：5 mins timeframe + 開 MACD
• 6 SMA
• 20 SMA
• MACD smoothing 9
• Allow premarket
入場規則（必須發生喺 key level 附近）：
• Short：快線（6）喺上面向下交叉慢線（20）就沽
• Long：快線（6）向上交叉慢線（20）就買
加勝率：如果 gap up 去 resistance，再等 620 chart 出 short entry signal，成功率通常更高（vice versa）。
這是最乾淨、最強勢的突破。

⭐ 8.3（修正版）：Stop Loss 不是任何 MA，而是「Setup Low + Buffer」
散戶 99% 做錯的事：
❌ 「我止蝕用 -5%」
 ❌ 「我用 10 日線破咗就走」
 ❌ 「我用 50MA 作止蝕」
全部錯。
專業方法只有一個：
⭐ 止蝕 = Setup Low（買入型態的最低點）+ 安全buffer
買入位置 → 有一個型態：
VCP


U&R


Pullback


Pivot Breakout


Range contraction


Trendline bounce


每個 setup 都一定有一個「防守低點」。
這低點是：
上一日低點（day low）


pivot low（突破前的低位）


cup-with-handle 的 handle low


VCP 中最後一段的低位


U&R 的 undercut low


⭐ 真正的止蝕，就是：
跌破這個 setup 的低點（+ 小 buffer）＝ 型態失效 → 必走。

✔ 每支股票波動不同
 ✔ setup 大小不同
 ✔ 範圍（range）不同
 ✔ beta 不同
 ✔ 行業不同
百分比止蝕完全是亂來。
專業交易者看的是：
⭐「型態是否破壞」
不是看跌了多少%。

⭐ 8.5：Initial Stop → Trailing Stop（簡易教材版）
買進後，你有兩個止蝕：

🔵 （1）Initial Stop（初始止蝕）
設定於：
Setup Low − buffer（1 ATR / 0.5 ATR）

🔵 （2）Trailing Stop（移動止蝕）
當價格升到你的方向後，你應移動止蝕：
方法可以是：
⭐（A）移至 Break-Even（保本）
→ 當升 1–2R
⭐（B）移至 Higher Low（更高的低點）
→ 以趨勢低點作防守
⭐（C）移至 MA（適用 trend-following）
→ 10MA / 21EMA / 50MA
⭐（D）移至 Swing Low
→ 以最近的回調低點作防守

⭐（1）勝率不重要，只需 EV > 0

每次贏 5R

⭐ 8.7：如何用止蝕計算倉位大小（你必須教家庭）
這個公式可以改變一生。

🔵 倉位 =（你願意輸的金額）÷（每股風險）
例：
你願意輸 $1,000
 Setup Low − Entry = 每股風險 $5
倉位 = 1000 / 5 = 200 股
就是這麼簡單。
這是專業交易員全部都用的公式。
